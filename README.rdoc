= Heirbnb

www.heirbnb.com

== Description


Airbnb is now available in Westeros! Check out the characters from Game of Thrones rent out their spaces.



== Project Information


Full stack web application built on Backbone.js and Ruby on Rails. Incorporates User, Rooms, Reviews, Bookings MVC. Custom google maps tiles and auto correction search functionality.
